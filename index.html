01
정적(Static) 콩팥 검사 DMSA SCINTIGRAPHY

임상적 적응증

  • 신우신염 진단 (콩팥 염증 소견)
  • 신장의 위치·크기·모양 평가
  • 말굽콩팥, 이소콩팥 확인

방사성의약품

  • 약품: ⁹⁹ᵐTc-DMSA
  • 집적기전: 능동이동
  • 집적부위: 세뇨관 (토리여과 거침)
  • 주사 후 1시간 내 투여량의 40~50%가 몸쪽 세뇨관에 결합
  • 콩팥겉질(피질, Cortex) 영상화
  • 모양 확인에 가장 좋음

검사 방법

DMSA 정맥주사
→
1~3시간 대기
→
방광 비우기
→
촬영
  • 조준기: 저에너지 고해상도 조준기
  • 자세: 바로 누운 자세 (검출기를 콩팥 중심에 위치)
  • 촬영 방향: ANT(전면상), POST(후면상), RPO 30°(우후면사위상), LPO 30°(좌후면사위상)
  • 소아: 1.5~2배 확대하여 영상 획득

✓ 후면상 (POST) — 기본

후벽장기 → 후면상이 가장 선명
일반적 콩팥 검사는 후면상 촬영

⚠ 전면상 (ANT) — 예외

말굽콩팥, 이소콩팥 (위치 이상) 확인 시 유용

영상 분석 & 판독

  • 콩팥겉질(신피질) 영상화 → 형태적 관찰 (위치·크기·모양)
  • 전면상·후면상 양쪽 콩팥에 ROI(관심영역) 설정 → 총 계수(total count) 산출
  • 정상 좌·우 비율: 44~56% (양측이 거의 동일)
📌 신우신염 소견
신우신염 → 염증 부위의 섭취 감소 → 영상에서 음성상(결손, 검게 갉아먹은 모양) 으로 나타남
02
동적(Dynamic) 콩팥 검사 기본 DTPA · MAG3 SCINTIGRAPHY

임상적 적응증

  • 콩팥 관류(혈류)와 기능 평가
  • 콩팥 세뇨관 기능 진단
  • 요로 폐쇄의 진단
목적
동적영상 = 신장 혈류상태 + 콩팥의 흡수·배설기능 평가

방사성의약품 핵심 비교

⁹⁹ᵐTc-DTPA

집적기전: 단순확산
집적부위: 토리(사구체)
측정: GFR

⁹⁹ᵐTc-MAG3

집적기전: 능동이동
집적부위: 세뇨관
측정: ERPF 표준
¹²³I/¹³¹I-Hippuran도 능동이동 → 세뇨관

전처치 ★★ 시험 빈출

① 수분 섭취
  • 검사 30~60분 전 물 300~500ml 음용
  • 요량이 2~3ml/min 이상 유지되도록
  • 수분 부족 시 DTPA가 신우(renal pelvis)에 오래 저류
② 검사 전 방광 비우기
  • 방광 충만 시 콩팥기능 지연 소견 초래
  • 방광 내 압력 증가 → 배설 장애
📌 수분섭취 + 방광비우기가 필요한 검사
① 콩팥 동적검사  |  ② Bone Scan  |  ③ PET 검사 (Torso 검사)

금지 약물 — 전처치 중단 기간 ★★★

약물 중단 시기 이유
이뇨제 (라식스, Lasix) 검사 전 1주일 콩팥 기능을 빠르게 도와줌 → 검사 결과 왜곡
캡토프릴 (Captopril) 검사 2일 전 콩팥 기능을 느리게 함 → 검사 결과 왜곡
조영제 최소 1주일, 가능하면 2주 후 콩팥 기능에 영향

검사 방법 & 영상 획득

  • 주사 방법: 순간정맥주사 (Bolus injection)
  • 자세: 바로 누운 자세, 검출기는 최소 콩팥과 방광 포함
  • 조준기: 저에너지 고분해능 평행구멍형 조준기
  • ⚠ ¹³¹I-hippuran만 고에너지 평행구멍형 조준기 사용!
  • 검출기 위치: 환자 등 뒤(후면) — 이식환자 제외!

콩팥성 고혈압 (신혈관성 고혈압) 진단

검사 방법 — 2번 실시

a. 기저스캔 (Baseline)

ACE억제제 투여 없이 검사
→ 저선량 MAG3 또는 DTPA 주사
→ 아무것도 안 넣고 동적검사

b. 캡토프릴 스캔

캡토프릴 25mg 또는 50mg 경구 투여
→ 투여 1시간 후 MAG3 또는 DTPA 투여
→ 검사 시행

결과 판독 ★★★ 핵심 로직

💡 핵심 원리
캡토프릴(ACE억제제) = 신장 기능 저하 물질
캡토프릴 먹으면 정상인은 사구체여과율이 저하되어야 한다!

✓ 정상 (고혈압 아님)

캡토프릴 투여 전후 차이가 크다
→ 약에 반응해서 기능이 잘 떨어짐
→ 콩팥 혈관은 정상

✗ 비정상 (콩팥성 고혈압)

캡토프릴 투여 전후 차이가 작다
→ 이미 기능 상실 → 변화 없음
→ 콩팥기능 상실 = 신혈관성 고혈압
⚡ 암기 공식
전후 차이 크면 → 정상  |  전후 차이 작으면 → 비정상 (콩팥기능 상실)
